Airbnb Data Cleaning Project

Overview

This project demonstrates data cleaning and preprocessing using Python and Pandas on the Airbnb Open Dataset.

Tasks Performed

- Removed unnecessary columns
- Renamed columns
- Removed duplicate records
- Handled missing values
- Cleaned price column
- Converted boolean values to numeric
- Exported cleaned dataset

Tools Used

Python
Pandas
Jupyter Notebook
